# Mini-app
ABC Mini-app is a prebuilt, hosted loyalty app optimized for customer engagement. It is a low-code integration that allows you to quickly create customizable customer engagement features to enable gamification on your website or application.

## Getting started
To get started with the ABC Mini-App integration, you can follow these steps:

* Learn how the Mini-App works
* Follow the quick start guide
* Customize the Mini-App to suit your needs
