# How the Mini-app Works
The Mini-app uses a client authentication flow to ensure that only authorized client software applications can access specific APIs or endpoints on the ABC Loyalty 3.0 platform. This helps prevent unauthorized access to customer data and services.

The client authentication flow involves obtaining the Client Key and Client Secret, making a POST request to the Client Authentication Endpoint provided by the SaaS platform, and using the One Time Token(OTT) to authenticate with the Mini-App Endpoint and open the Mini-App for its users.
