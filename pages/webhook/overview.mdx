# Event Webhook
In this guide, you will learn how the webhook works, including an overview of the event payload flow. You will also find a quickstart section that includes details on how to obtain an API key and webhook endpoint, and how to send data payloads to the ABC Loyalty 3.0 server. 

Additionally, you will find examples of the JSON data format for the event payload and responses you may encounter when sending your data payloads. 

Let's get started!
