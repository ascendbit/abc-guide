# Data Publisher

The Data Publisher feature allows you to seamlessly send events generated within our system to third-party services for further analysis and utilization. By integrating with popular services like Google Analytics 4, Mixpanel, and Braze, you can gain valuable insights and enhance your marketing and analytics strategies.

![Data Publisher](/dashboard-publisher.png)

This section of the developer guide provides step-by-step instructions on setting up the integration, testing the connection, selecting events to be sent to each publisher, and managing the streaming process. Please refer to the respective sub-pages for detailed instructions specific to each supported publisher.
